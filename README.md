## 介绍
本仓库为nonebot-plugin-hammer-xxx等插件的核心依赖，其中主要内容为我本人（ArgonarioD）在开发nonebot2插件与bot时常用的轮子，本仓库在一般情况下不作为nonebot的插件而是作为插件的依赖库安装到nonebot的运行环境中

## 已经实现的轮子
 - 常量 util/constant.py
 - 快速构建消息（回复消息） util/message_factory.py
